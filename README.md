# More Garlic 🧄 — website

Public marketing + legal site for the iOS app **More Garlic**
(bundle id `com.langberg.moregarlic`, App Store id `6782392934`).

Static HTML/CSS, no build step — served directly by GitHub Pages.

| Page | URL |
| --- | --- |
| Landing | https://negativetime.github.io/more-garlic-site/ |
| Privacy Policy | https://negativetime.github.io/more-garlic-site/privacy.html |
| Support | https://negativetime.github.io/more-garlic-site/support.html |

The Privacy and Support URLs are used in the App Store listing.

## Files
- `index.html` — landing page (hero, features, privacy callout)
- `privacy.html` — Privacy Policy
- `support.html` — Support / FAQ
- `style.css` — shared stylesheet (mobile-first, accessible, light/dark aware)

## Brand
"Playful Pantry": aubergine/purple accent (`#5b2a86`), garlic 🧄, warm off-white
background, rounded system font stack. Tagline: *Turn any recipe into a smart
grocery list.*
